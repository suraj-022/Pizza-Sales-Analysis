<h1 align="center">🍕 Pizza Sales Data Analysis</h1>

<h2>📌 Project Overview</h2>
<p>
This project presents a comprehensive <b>data analysis of pizza sales</b> using Python. 
The main objective is to extract meaningful insights from the sales dataset in order to 
understand revenue patterns, customer ordering behavior, and product performance.
</p>

<p>
Through <b>Exploratory Data Analysis (EDA)</b> and visualization techniques, the project 
helps identify key trends that can assist restaurant owners or managers in making 
<b>data-driven business decisions</b>, improving menu strategies, and optimizing sales performance.
</p>

<h2>🎯 Business Objectives</h2>

<ul>
<li><b>Calculate Key Performance Indicators (KPIs)</b> such as:
    <ul>
        <li>Total Revenue</li>
        <li>Total Pizzas Sold</li>
        <li>Total Orders</li>
        <li>Average Order Value</li>
        <li>Average Pizzas per Order</li>
    </ul>
</li>

<li><b>Analyze Sales Distribution</b> across different dimensions including:
    <ul>
        <li>Pizza Categories</li>
        <li>Pizza Sizes</li>
        <li>Individual Pizza Names</li>
    </ul>
</li>

<li><b>Identify Sales Trends</b> by analyzing:
    <ul>
        <li>Peak Sales Days</li>
        <li>Peak Sales Hours</li>
    </ul>
</li>

<li><b>Menu Optimization</b> by identifying:
    <ul>
        <li>Top-selling pizzas</li>
        <li>Low-performing or underperforming pizzas</li>
    </ul>
</li>

</ul>

<h2>🛠️ Tools & Technologies Used</h2>

<ul>
<li><b>Python</b> – Core programming language used for data analysis.</li>
<li><b>Pandas</b> – Data cleaning, transformation, and manipulation.</li>
<li><b>Matplotlib</b> – Creating statistical and analytical visualizations.</li>
<li><b>Seaborn</b> – Advanced and aesthetically pleasing data visualizations.</li>
</ul>

<h2>📂 Dataset Source</h2>

<p>
The dataset used for this analysis was obtained from 
<b>Kaggle</b>. It contains transactional pizza sales data including 
order details, pizza categories, sizes, prices, and timestamps.
</p>

<h2>📊 Analysis Workflow</h2>

<ul>

<li><b>Data Loading</b> – Importing the dataset and understanding its structure.</li>

<li><b>Data Cleaning</b> – Handling missing values, correcting data types, and preparing the dataset for analysis.</li>

<li><b>KPI Calculation</b> – Computing important business metrics such as revenue, order counts, and sales averages.</li>

<li><b>Exploratory Data Analysis (EDA)</b> – Exploring the dataset to uncover trends and patterns in sales.</li>

<li><b>Data Visualization</b> – Using charts such as:
    <ul>
        <li>Bar Charts</li>
        <li>Trend Line Charts</li>
        <li>Heatmaps</li>
        <li>Category Distribution Charts</li>
    </ul>
</li>

<li><b>Insights & Conclusions</b> – Interpreting the results to generate meaningful business recommendations.</li>

</ul>

<h2>📈 Key Insights</h2>

<ul>
<li>Identified the most popular pizza categories and best-selling products.</li>
<li>Detected peak ordering hours and busiest sales days.</li>
<li>Highlighted menu items with low performance that may require improvement or removal.</li>
<li>Provided insights that can support better pricing and menu strategies.</li>
</ul>
